**Q. 박재형**
```
(1) : if (nullValue) {
  // ...
}

(2) : if (nullValue !== null) {
  // ...
}
```
1번, 2번 코드의 성능적 차이가 있는지?

**A. 장민호**
1번 코드는 Truty, Falsy를 판단하는 연산과정이 한번 있고, 2번은 바로 비교를 하기 때문에 성능이 최대 3% 차이가 난다.
